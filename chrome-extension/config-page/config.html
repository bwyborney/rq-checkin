CPR Check-In Configuration

Select a device type, then choose what you'd like to appear on your check-in form for that device type.

Under "Pre-tests," type the name of the component in the first column. For example, "Face ID." In the second column, choose the possible outcomes of this test. If you choose "amps" or "percent," the pre-test form will give you a box to type in a number, then it will add a percent symbol or "amps" to the end of what you typed.

Under "Repairs," enter all the repairs you might want to perform on this device type. In the first column, choose "Yes/No" for a single item with a checkbox. Choose "Pick one" for repairs where you need to select one of multiple options (think "OEM screen," "aftermarket screen"). Choose "Other" to add an "other" option which will let you manually enter a repair every time you fill out this form.
